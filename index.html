PCMRBot!

PCMRBot is a Twitch chat bot that adds polls, raffles, automatic replies, and a few other commands. We am always looking to improve the bot so we made it Open Source! Visit the Github repo and send something our way! To get the bot to join your channel, visit the PCMRBot channel and type !join! Type !leave in YOUR channel when you are ready for it to leave!.

Polls

Ever wanted to poll your chat on something? Do you find creating a Straw Poll too annoying? Then use the !poll command! Set a time limit for the poll and it will keep track of the votes and even allows user to vote for a random option if they aren't sure what to choose! More info on the !poll command can be found here!

Automatic Replies

Tired of getting the same question over and over again? Wish there was an easy way to answer without telling them to type a command? Introducing Auto Replies! Tell the bot what keywords to look for and if it finds all of them then it will send a message in chat, its that easy! More info on the command can be found here!

Ideas

Have any other ideas for the bot? Visit the github repo and add it! If we approve we'll pull it in! Don't know how to code? Send an email to pcmrbot7@gmail.com with the subject "PCMRBot Feature Request" and we'll see what we can do!

Raffles

Have something to give away, but only want subscribers or followers to be allowed to participate? Use the !raffle command and it will choose a random person in the chat who meets the requirements you provide! More info on the !raffle command can be found here!
